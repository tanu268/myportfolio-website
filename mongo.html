My MongoDB Learning Journey: Why I Started

When I began my data science journey, I quickly realized that understanding databases is crucial. One of the most popular and powerful NoSQL databases today is MongoDB — a flexible, document-based database that can handle complex, nested data structures easily.

My mentor recommended that I learn MongoDB early on because it’s widely used in real-world projects and modern applications, especially those involving big data and scalable architectures. This pushed me to take the first step and start learning it in depth.

Day 1 of Learning MongoDB

Date: 03 June

On my first day, I officially kicked off my MongoDB journey! I installed the two main tools needed to work with MongoDB effectively:

  • MongoDB Compass — a GUI tool that makes it easier to visualize and manage your data.
  • MongoDB Shell (mongosh) — a command-line tool for running database commands.

I connected to my local MongoDB server using the shell with this command:

mongosh "mongodb://localhost:27017"

Then I started practicing the basics:

  • Creating databases and collections
  • Inserting documents, including nested (embedded) documents
  • Updating fields, including arrays and embedded objects

Here are some example commands I used:

use latest_db  
db.students.insertOne({ name: "Ram", age: 12 })  
db.students.updateOne({ name: "shyam" }, { $set: { idCards: { hasPanCard: false, hasAdhaarCard: true } } })  
db.students.updateMany({}, { $set: { hobbies: ["Anime", "Cooking"] } })
Fun fact: MongoDB supports nesting documents up to 100 levels deep and stores data up to 16MB per document, which is super powerful for complex data modeling.

Learning Resources

To guide my learning, I followed:

  • W3Schools MongoDB tutorial — great for clear, step-by-step documentation and examples.
  • Engineering Digest YouTube Channel — offers practical walkthroughs and real-world use cases that helped me understand MongoDB’s concepts better.
What’s Next?

This is just the beginning of my MongoDB journey! I plan to keep practicing queries, exploring advanced features like aggregation, indexing, and replication, and sharing my progress daily.

If you’re new to MongoDB or want to strengthen your basics, follow along with me!

The path to mastery is paved with curiosity and courage. Don’t be afraid to explore the unknown, ask questions, and make mistakes. Each challenge you face is an opportunity to grow stronger and wiser. Keep your passion alive and trust the process — success will follow.
