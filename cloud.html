Getting Started with Cloud

My first formal introduction to cloud computing came through my college, which enrolled us in two major aws cloud courses: AWS Learner Lab and AWS Cloud Foundations. Both courses opened the door to a whole new world of technology that powers so many apps and websites today.

AWS Learner Lab: Hands-On with EC2 and More

The AWS Learner Lab was an exciting start because it gave me real, hands-on experience with cloud services. One key component I learned about was EC2 (Elastic Compute Cloud).

EC2 is basically like renting a virtual computer in the cloud. Instead of buying a physical server, you can launch and manage virtual servers on-demand. This means you can run applications, host websites, or even experiment with data processing, all without needing your own hardware.

Learning how to set up EC2 instances, configure security groups, and manage resources gave me practical insights into how scalable and flexible cloud computing really is.

Cloud Foundations: Building the Basics

The other course, Cloud Foundations, was focused on understanding the core concepts of cloud computing ‚Äî the what, why, and how behind the technology.

I completed 4 modules in this course over the span of a year. The course helped me grasp the essential building blocks, like cloud deployment models, service models (IaaS, PaaS, SaaS), and key AWS services beyond EC2.

Our college re-enrolled us for this course after the first year, so I had another chance to deepen my knowledge and stay updated.
