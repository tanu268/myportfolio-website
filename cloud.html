Journey to the Cloud: Unlocking Cloud Computing

  • By Tanu Namdeo

My Cloud Learning Journey: How It All Started

Hello everyone, Tanu here! Today, I want to share the story behind my dive into cloud computing — how I got introduced to it and why it became a key part of my learning path.

Getting Started with Cloud

My first formal introduction to cloud computing came through my college, which enrolled us in two major aws cloud courses: AWS Learner Lab and AWS Cloud Foundations. Both courses opened the door to a whole new world of technology that powers so many apps and websites today.

AWS Learner Lab: Hands-On with EC2 and More

The AWS Learner Lab was an exciting start because it gave me real, hands-on experience with cloud services. One key component I learned about was EC2 (Elastic Compute Cloud).

EC2 is basically like renting a virtual computer in the cloud. Instead of buying a physical server, you can launch and manage virtual servers on-demand. This means you can run applications, host websites, or even experiment with data processing, all without needing your own hardware.

Learning how to set up EC2 instances, configure security groups, and manage resources gave me practical insights into how scalable and flexible cloud computing really is.

Cloud Foundations: Building the Basics

The other course, Cloud Foundations, was focused on understanding the core concepts of cloud computing — the what, why, and how behind the technology.

I completed 4 modules in this course over the span of a year. The course helped me grasp the essential building blocks, like cloud deployment models, service models (IaaS, PaaS, SaaS), and key AWS services beyond EC2.

Our college re-enrolled us for this course after the first year, so I had another chance to deepen my knowledge and stay updated.

Exploring Google Skill Boost Cloud

Alongside AWS, I also enrolled in Google Skill Boost Cloud, a paid program (₹500 annually) that offers extensive cloud learning material.

Starting with Cloud Foundations on Google’s platform, I found it packed with interactive labs, quizzes, and real-world scenarios. It’s been a great complement to my AWS studies because it broadens my understanding of cloud concepts across different providers.

Why Cloud Computing?

Cloud computing has become an essential skill for anyone in tech today. It’s not just about hosting websites or running apps — it’s about understanding how to build scalable, secure, and efficient systems in a fast-changing digital world.

My journey into cloud started with curiosity and grew into a passion, thanks to these courses and hands-on experiences. I’m excited to keep learning and exploring more cloud technologies in the future.

Thank you for reading! I hope my story inspires you to take that first step into cloud computing too. Stay tuned for more updates on my learning journey.

Success is not about giant leaps, but about taking small, consistent steps forward every day. Keep learning, stay curious, and never be afraid to make mistakes — they are the stepping stones on your journey to mastery.
