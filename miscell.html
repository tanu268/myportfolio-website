Introduction

Today I finally solved a real-world Python problem that I had once failed ‚Äî calculating a monthly in-hand salary after deductions and tax.

At first, it seemed easy: input salary, apply some deductions, and display the final amount. But soon, I realized the logic behind salary ranges and percentage deductions needed more attention than I expected.

The Problem

Write a Python program that takes a user's annual CTC (Cost To Company) and calculates the monthly in-hand salary after:

  • HRA deduction (10%)
  • DA deduction (5%)
  • PF deduction (3%)
  • Income tax deduction based on CTC:
CTC in ‚Çπ Lakhs    ‚Üí    Tax Rate
Below 5           ‚Üí    0%
5 to 10           ‚Üí    10%
10 to 20          ‚Üí    20%
Above 20          ‚Üí    30%

My First Attempt

I started by writing:

ctc = int(input("Enter your salary"))
deductions = ctc * (0.1 + 0.05 + 0.03)
if ctc < 500000:
    tax = 0
elif ctc <= 500000 and ctc > 100000:
    tax = deductions * 0.1
...

But I kept getting wrong results. I realized my mistake was:

  • I was applying tax on the deduction, not on the remaining salary.
  • My conditional checks like ctc <= 500000 and ctc > 100000 were logically confusing.

What Confused Me Most: Conditional Ranges

The line I struggled with most was:

elif salary <= 500000 and salary > 100000:

Even though it works, it feels mentally hard to parse. I learned that Python supports chained comparisons, which are much easier to read:

elif 100000 < salary <= 500000:

This line is simpler and matches how we speak: ‚Äúsalary is between 1L and 5L.‚Äù

Final Working Code

# Program to calculate in-hand monthly salary

ctc = float(input("Enter your annual CTC in ‚Çπ: "))

# Deductions
hra = ctc * 0.10
da = ctc * 0.05
pf = ctc * 0.03
fixed_deductions = hra + da + pf

# Salary after fixed deductions
after_deductions = ctc - fixed_deductions

# Tax calculation
if ctc <= 500000:
    tax = 0
elif 500000 < ctc <= 1000000:
    tax = after_deductions * 0.10
elif 1000000 < ctc <= 2000000:
    tax = after_deductions * 0.20
else:
    tax = after_deductions * 0.30

# Final in-hand salary
in_hand_annual = after_deductions - tax
monthly_salary = in_hand_annual / 12

print("Your in-hand monthly salary is: ‚Çπ", round(monthly_salary, 2))

What I Learned

  • Break problems into steps: input ‚Üí deductions ‚Üí tax ‚Üí monthly output.
  • Avoid overcomplicating if-else conditions.
  • Always double-check if the logic matches the tax slabs.
  • Comment your code ‚Äî it helps you later.
  • Python‚Äôs range chaining like min < x <= max is super useful and clear.
