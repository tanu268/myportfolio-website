Bug I Found in a More Advanced Version:

Infinite loops happen if input and counter increment are inside the wrong block. Here's the corrected snippet:

import random

jackpot = random.randint(1, 100)
guess = int(input("Guess the number: "))
counter = 1

while guess != jackpot:
    if guess < jackpot:
        print("Wrong! Guess higher.")
    else:
        print("Wrong! Guess lower.")
    guess = int(input("Guess again: "))
    counter += 1

print("Correct guess!")
print("Attempts:", counter)
